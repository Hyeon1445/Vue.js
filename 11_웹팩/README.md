## 웹팩

* 앵귤러, 리엑트, 뷰에서 권하는 모듈 번들러

* 서로 연관이 있는 모듈 간의 관계를 해석하여 정적인 자원으로 변환해주는 변환 도구

* 파일 간의 연간 관계를 파악하여 __하나의 자바스크립트 파일로__ 변환해 주는 변환 도구

* 한 개의 파일에 담는 이유 ) 웹/앱 로딩속도 향상
