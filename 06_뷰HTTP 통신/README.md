# HTTP 통신

HTTP) 브라우저와 서버 간에 데이터를 주고받는 통신 프로토콜

* 브라우저에서 특정 데이터를 보내달라고 request
* response로 해당 데이터를 보내 준다

리엑트, 앵귤러 등 -> ajax 사용

뷰 -> ajax를 지원하기 위한 라이브러리 제공 : 뷰 리소스 & axios

***

## 뷰 리소스

사용 방법 1) CDN을 이용해서 라이브러리 로딩
사용 방법 2) NPM으로 라이브러리 설치(ES6)
